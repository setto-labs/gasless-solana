# Setto - Solana Program

Anchor program for gasless token transfers on Solana.

## Overview

Enables gasless token transfers using:
- **Server Signer** - Authorized signers for transaction verification
- **Relayer Network** - Whitelisted relayers pay gas on behalf of users
- **SPL Token Support** - Native support for USDC, USDT and other SPL tokens
